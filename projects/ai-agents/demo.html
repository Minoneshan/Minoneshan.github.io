AlphaAgents Demo

Multi-Agent System Architecture

Walkthrough of the specialist agent debate loop, data pipeline, and portfolio construction workflow.

Specialist Agent Flow

1. Data Hydration

Goal: Fetch market data, SEC filings, and news articles for a stock universe.

python -m apps.data_fetch \
  --tickers AAPL MSFT META \
  --period 3mo \
  --save-artifacts

Output: Cached market history, filing documents, headline summaries stored in local artifacts.

2. Specialist Analysis

Goal: Run independent agents (fundamental, valuation, sentiment) on the hydrated data.

Fundamental Agent: 
python -m apps.fundamental --ticker AAPL

Reads 10-K/10-Q via RAG, prompts LLM for BUY/HOLD/SELL opinion on business fundamentals.

Valuation Agent: 
python -m apps.valuation --ticker AAPL

Computes technical indicators (momentum, RSI), price mean-reversion, relative valuation metrics.

Sentiment Agent: 
python -m apps.sentiment --ticker AAPL --days 7

Aggregates news headlines, computes recency-weighted sentiment, produces consensus narrative.

3. Debate & Consensus

Goal: Collaborator agent orchestrates debate, synthesizes specialist opinions into portfolio signal.

python -m apps.portfolio \
  --tickers AAPL MSFT META AMZN GOOG \
  --portfolio-strategy v3 \
  --period 3mo

Output Format (per ticker):

{
  "ticker": "AAPL",
  "final_signal": "BUY",
  "confidence": 0.78,
  "fundamental_opinion": "Strong BUY",
  "valuation_score": 0.65,
  "sentiment_summary": "Positive momentum, earnings optimism",
  "position_weight": 0.22
}

4. Backtesting & Evaluation

Goal: Convert agent signals into portfolio returns, compute performance metrics.

python -m apps.evaluation \
  --tickers AAPL MSFT META AMZN GOOG \
  --portfolio \
  --export-dashboard

Performance Metrics:

  • Cumulative returns & Sharpe ratio
  • Drawdown analysis & recovery metrics
  • Agent decision frequency & consensus rates
  • Per-ticker contribution to portfolio risk

5. Human-in-the-Loop Review

Goal: Analyst reviews decisions, approves or rejects with overrides logged for RLHF.

python -m apps.human_interface \
  --tickers AAPL MSFT META \
  --reviewer "Analyst Name"

Feedback Capture: Each override is logged as JSON with original signal, revised opinion, and reasoning. Perfect for building RLHF datasets to improve agent calibration.

Interactive Dashboard

A Streamlit dashboard is available to visualize agent decisions, debate frequency, and portfolio construction in real-time.

streamlit run dashboard/app.py

Displays selected tickers, agent rationales, guardrail checks, and contribution to portfolio risk.

Flexible Configuration

The system is designed for plug-and-play customization without code changes:

LLM Provider Selection

Ollama-Only 
export FUNDAMENTAL_PROVIDER=ollama
export SENTIMENT_PROVIDER=ollama
export OLLAMA_MODEL=mistral

Fully local, cost-friendly.

OpenAI-Only 
export OPENAI_API_KEY=sk-...
export OPENAI_MODEL=gpt-4o
export FUNDAMENTAL_PROVIDER=openai

Premium quality, managed latency.

Hybrid 
FUNDAMENTAL_PROVIDER=openai
SENTIMENT_PROVIDER=ollama

Per-agent optimization.
