AlphaAgents Multi-Agent Research Platform

Production-grade multi-agent LLM framework for quantitative equity research, fundamental analysis, and portfolio construction. Inspired by the AlphaAgents paper, this system orchestrates specialist agents (fundamental, valuation, sentiment) to debate and synthesize trading signals with human-in-the-loop oversight.

System Overview Full Documentation View on GitHub

The Challenge

Modern quantitative investing requires synthesizing disparate data sources—SEC filings, market prices, news sentiment—into actionable portfolio decisions. Traditional approaches rely on rigid scoring systems. This project implements an LLM-powered debate mechanism where specialist agents argue about each stock, reaching consensus via a collaborator orchestrator.

Key Innovation: Specialist debate loop where fundamental, valuation, and sentiment agents reach consensus on BUY/HOLD/SELL, with human-in-the-loop guardrails and RLHF-ready feedback capture.

System Architecture

7-Layer Architecture

  1. Data Layer – Market data, SEC filings, news ingestion
  2. Agent Layer – Fundamental, valuation, sentiment specialists
  3. Orchestration – Debate loop, consensus building
  4. Backtesting – Signal-to-returns conversion
  5. Evaluation – Performance metrics & dashboards
  6. Human Feedback – Analyst overrides & RLHF logging
  7. Application – CLIs & Streamlit dashboards

Key Features

  • Multi-Agent Debate – Collaborator orchestrates specialist consensus
  • RAG-Powered Analysis – LLamaIndex over SEC filings
  • Flexible LLM Providers – Ollama, OpenAI, or hybrid
  • Human-in-the-Loop – Analyst overrides & feedback capture
  • Full Backtesting – Single & multi-ticker portfolio analysis
  • Modular Design – Independent specialist development
