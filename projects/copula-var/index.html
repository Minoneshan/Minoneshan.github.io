Copula-Tail VaR Engine

A tail-risk engine that combines extreme value theory with t-copulas to estimate VaR and Expected Shortfall for multi-asset portfolios.

Launch Demo Read Documentation View GitHub

Problem

Gaussian VaR understates the probability of joint extreme losses. This engine models marginal tails with EVT and joint tail dependence with a Student-t copula, producing more realistic portfolio-level tail-risk estimates.

Methods

  • Peaks-over-threshold EVT (Generalized Pareto tails)
  • Student-t copula for joint tail dependence
  • Monte Carlo VaR and Expected Shortfall
  • Backtesting via Kupiec and Christoffersen tests

Outputs

  • Portfolio VaR and ES at multiple confidence levels
  • Marginal tail diagnostics
  • Backtest exception report
