CPI Transformer Risk Dashboard

A transformer-based model for forecasting inflation, paired with an explainability dashboard that surfaces the macro drivers behind each forecast.

Launch Demo Read Documentation View GitHub

Problem

Inflation surprises move markets, but black-box forecasts are hard to trust. This project pairs a strong sequence model with explainability so the drivers of each forecast are transparent.

Methods

  • Transformer encoder over macro time series
  • Multi-horizon CPI forecasting
  • Attention and SHAP-based attribution
  • Backtesting against survey consensus

Outputs

  • Point and interval inflation forecasts
  • Driver attribution dashboard
  • Forecast error diagnostics
