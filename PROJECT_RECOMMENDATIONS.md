## 📋 How to Add New Projects

### Step 1: Create Project Folder Structure
```bash
mkdir -p projects/trading-agents
mkdir -p projects/waterbridge-mm
```

### Step 2: Create Files for Each
For each new project, create 3 files:
- `projects/trading-agents/index.html` — Project landing page
- `projects/trading-agents/demo.html` — Demo/walkthrough
- `projects/trading-agents/docs.html` — Technical documentation

### Step 3: Add to projects.html
Add new project cards to the projects.html grid with filters. Use this template:

```html
<!-- Project: Trading Agents -->
<article class="project-card" data-tags="ml optimization">
  <h3>
    <a href="projects/trading-agents/index.html">Trading Agents</a>
  </h3>
  <p class="desc">
    Multi-agent LLM framework for quantitative trading research and execution.
  </p>
  <div class="tags">
    <span class="tag">Python</span>
    <span class="tag">LLM</span>
    <span class="tag">Multi-Agent</span>
    <span class="tag">Machine Learning</span>
  </div>
  <div class="card-actions">
    <a href="projects/trading-agents/demo.html" class="btn btn-secondary btn-sm">Demo</a>
    <a href="projects/trading-agents/docs.html" class="btn btn-secondary btn-sm">Documentation</a>
    <a href="https://github.com/Minoneshan/TradingAgents" class="btn btn-ghost btn-sm" target="_blank" rel="noopener">GitHub</a>
  </div>
</article>
```

### Step 4: Add Filter Tags
Update the filter buttons in projects.html:
- Current: `[All] [Risk] [Derivatives] [Machine Learning] [Macro] [Optimization] [C++]`
- Add: `[LLM]` or use existing `[Machine Learning]` tag
